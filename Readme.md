# interact-repl
Simple repl in zig 

## Build and run
```sh
# Simple af since it is zig
zig build run
```

# Usage
```sh
\\ Usage
\\  Current result: {d}
\\  Commands
\\    help, usage - Prints this message
\\    quit, exit - Duh
\\    add n - Adds a number to the last result
\\    result - Prints the last result
```
